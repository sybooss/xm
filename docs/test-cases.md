# 测试用例清单

## 单轮问答样例（20 条）
1. 我的蓝牙耳机不想要了，可以退货吗？
2. 数码产品拆封以后还能不能退？
3. 我这单衣服尺码不合适，想换大一码。
4. 换货是谁承担运费？
5. 退款一般多久能到账？
6. 银行卡退款失败怎么办？
7. 商家 24 小时还没审核，我该怎么办？
8. 物流两天没动静，可以催件吗？
9. 快递显示已签收，但我没收到。
10. 订单发货前能直接取消吗？
11. 已经发货了，还能改收货地址吗？
12. 我直接拒收包裹后会马上退款吗？
13. 退货的时候赠品要不要一起寄回？
14. 部分退款以后优惠券会退回吗？
15. 退货以后积分怎么处理？
16. 我买的是生鲜，能不能七天无理由退货？
17. 家电已经安装了，还能退吗？
18. 我想问一下这个商品是不是预售。
19. 商家一直不处理，能帮我转人工吗？
20. 我想投诉物流把东西弄丢了。

## 多轮对话样例（10 组）
1. 先问“蓝牙耳机可以退货吗” -> 再问“那退款多久到账？”
2. 先问“衣服尺码不合适想换货” -> 再问“那现在怎么办？”
3. 先问“快递一直没更新” -> 再问“如果还是不动可以投诉吗？”
4. 先问“订单发货前能取消吗” -> 再问“取消后钱什么时候退？”
5. 先问“包裹破损签收了怎么办” -> 再问“需要什么照片？”
6. 先问“商家超时不处理” -> 再问“多久能转人工？”
7. 先问“赠品没了还能退吗” -> 再问“那会扣多少钱？”
8. 先问“组合商品想退一部分” -> 再问“优惠券会退回吗？”
9. 先问“预售商品什么时候发货” -> 再问“如果不想等了可以提现吗？”
10. 先问“银行卡退款失败” -> 再问“能不能换一张卡退？”

## 重点核验项
- 每条回复都要有意图标签和置信度。
- 搜索结果要能命中对应规则文档。
- 多轮追问要体现上下文承接，不要每次都像新会话。
- 无 OpenAI Key 时，系统仍能本地稳定回答并完成演示。
